## Terraform Provider

[Terraform](https://www.terraform.io) is an infrastructure as code software to automate the deployment of infrastructure resources.

Scalingo maintains a [Terraform provider](https://registry.terraform.io/providers/Scalingo/scalingo/latest/docs) that you may want to use to automate the deployment and configuration of your Scalingo resources.

### Get started

To start using the Scalingo Terraform provider:

1. [Download and install Terraform](https://developer.hashicorp.com/terraform)
2. Create a Scalingo [API tokens]({% post_url platform/user-management/scalingo-account/2000-01-01-navigating %}#api-tokens) from your account settings
3. Export your secrets and configuration as Environment Variables

For the complete list of resources, schema details and examples, see the provider [documentation page](https://registry.terraform.io/providers/Scalingo/scalingo/latest/docs).
