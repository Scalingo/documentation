title: ‘Increasing the maximum application image size to 2 GiB’
---

The maximum size of an application image is now **2 GiB**.

We still recommend keeping your application image as small and lightweight as possible. Smaller images help make builds, deployments, and application scaling faster and smoother.

If your application image exceeds this limit, you will need to reduce its size before deployment.
