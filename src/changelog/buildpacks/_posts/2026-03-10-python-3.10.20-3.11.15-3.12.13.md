Python `3.10.20`, `3.11.15`,  and `3.12.13` are now available. They are also
the default versions for, respectively, Python 3.10, Python 3.11, and Python
3.12.

Changelog:
- [Python 3.12.13](https://docs.python.org/3.12/whatsnew/changelog.html#python-3.12.13-final)
- [Python 3.11.15](https://docs.python.org/3.11/whatsnew/changelog.html#python-3.11.25-final)
- [Python 3.10.20](https://docs.python.org/3.10/whatsnew/changelog.html#python-3.10.20-final)
