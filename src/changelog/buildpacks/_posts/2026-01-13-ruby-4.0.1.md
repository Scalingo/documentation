Ruby `4.0.1` is now available.

Bundler version installed now directly exactly matches the value in
`BUNDLED WITH` from the `Gemfile.lock` file. Previously, this value was
converted to a "known good version."

Applications without a `BUNDLED WITH` value will receive a default bundler
version.

[Changelog](https://www.ruby-lang.org/en/news/2026/01/13/ruby-4-0-1-released/)
