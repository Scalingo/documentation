Node.js `25.5.0` is now available.

The corepack tool is no longer used to install pnpm and Yarn. Usage of the
`packageManager` field in `package.json` to declare pnpm as the package manager
is still supported.

Changelogs:
- [Node.js 25.5.0](https://github.com/nodejs/node/blob/main/doc/changelogs/CHANGELOG_V25.md#25.5.0)
