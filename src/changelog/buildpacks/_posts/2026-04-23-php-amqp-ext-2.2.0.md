PHP extension `amqp` `2.2.0` is now available for PHP >= 8.2 on both
`scalingo-22` and `scalingo-24`.

The extension is built against `librabbitmq` `0.15.0`.

Changelogs:
- [amqp 2.2.0](https://pecl.php.net/package/amqp/2.2.0)
- [librabbitmq 0.15.0](https://github.com/alanxz/rabbitmq-c/releases/tag/v0.15.0)
