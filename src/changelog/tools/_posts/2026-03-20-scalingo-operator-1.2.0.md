The [Scalingo Operator](https://github.com/Scalingo/scalingo-operator) can deploy and undeploy PostgreSQL instances hosted on dedicated resources on Scalingo platform from a [Kubernetes](https://kubernetes.io/) cluster.

Main features:

* deploy/undeploy PostgreSQL instances hosted on Scalingo dedicated resources
* add and remove firewall rules
* apply plan updates
