Scalingo supports the following versions of Nginx:

| Nginx version         | `scalingo-20` (EOL)  | `scalingo-22`  |
| --------------------- | -------------------- | -------------- |
| **`1.28`** (Stable)   | Unsupported          | Up to `1.28.0` |
| **`1.27`** (Mainline) | Up to `1.27.5`       | Up to `1.27.5` |
| **`1.26`**            | Up to `1.26.3`       | Up to `1.26.3` |
